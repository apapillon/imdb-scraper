IMDB-scraper
------------
Permet de récupérer la liste des films dans lesquels à jouer un acteur passé
en paramètre.

Dépendances
-----------
- [BeautifulSoup][1]

Utilisation
-----------
        python imdb-scraper/imdb-scraper.py Charles Chaplin

[1]: http://www.crummy.com/software/BeautifulSoup/
